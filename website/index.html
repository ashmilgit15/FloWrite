CAPABILITIES

Engineered for Speed & Precision

Real-Time Transcription

Sub-second voice-to-text powered by Groq's Whisper Large V3 Turbo. Speak naturally, get perfect text.

Floating Overlay

A draggable mic button that floats over every app. Tap, speak, and text auto-fills into any input field.

Smart Injection

Accessibility-based text injection pipes dictated text directly into focused fields — no clipboard hack needed.

Semantic Cleanup

Wispr Flow-inspired post-processing removes filler words, handles self-corrections, and auto-capitalizes.

Privacy First

API keys compiled into the build — never transmitted. Audio processed via API, never stored on servers.

AMOLED Optimized

Pure pitch-black UI with neon cyber accents. Designed for OLED screens — zero wasted pixels, zero wasted battery.

WORKFLOW

Three Taps to Flow State

01

Launch Overlay

Tap "Start Overlay Service" from the home screen. A floating mic button appears on top of every app.

02

Tap & Speak

Tap the floating mic. Speak naturally — FloWrite listens with intelligent voice activity detection and auto-stops.

03

Text Appears

Clean, polished text is instantly injected into the focused input field. No paste, no switch — just flow.
